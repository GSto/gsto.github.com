Portfolio

Wagglez iPhone App

Wagglez Wagglez is a mobile application available for the iPhone That promotes local deals in the Athens, GA Area. Wagglez was designed to kill the coupon book. It allows local businesses to post and manage their coupons online, and allow users the easily redeem with their phone. The platform also provides analytics that aren’t possible with traditional paper coupons. The app was built using the Alloy framework for Titanium , And works with a RESTful API built in Ruby on Rails.

Welcome Home WordPress Template

The Welcome Home WordPress template is a work-in-progress WordPress template designed for Real Estate and leasing agencies. The system was designed to both help feature properties, and provide an SEO-optimized blogging system so that real estate agencies can build their brand and get more leases with content marketing. The design is clean, minimal, and focused, to bring the properties themselves into focus. The design is also fully responsive, enabling people to view your site and find properties on the go, including the ability for people to find home near them using tools like the HTML5 geolocation API. This theme will be available for sale in the near future.

Radiology Information System

This RIS(Radiology Information System) Is an internal web application currently under development for a client. The goal of the tool is to increase inter-office communication and productivity. There are several modules in the system to support this, including a patient database, an appointment calendar, and a system that replaces the arduous process of patients filling out paper forms with a responsive, cloud-based from optimized for use with an iPad. With a design focused on ease of use and simplicity, this RIS tool makes lives easier for patients, office staff, and doctors alike. This application was built using the Foundation CSS library and the CodeIgniter framework.

Office of International Education

international.uga.edu

Website for the University of Georgia’s Office of International Education. The office manages study abroad programs as well as exchange students from foreign countries. The site is highly complex, as it contains many resources for both students, faculty, and partner universities. To support this, the site is built with easy-to-use page editing and navigation editing tools, giving the client an exceptional amount of control over their content. This project is something I worked on while employed for JH Media Group. All design work was done by the design team, not by me. And while I worked on the programming back end, the site is built on J House Media’s TigerAdmin framework.
